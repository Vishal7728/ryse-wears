4. HOW TO SETUP
===============
Step 1: Place your images in the appropriate folders with exact names listed above
Step 2: Run the database setup script: node createWesternProducts.js
Step 3: Start your backend server: npm start (in backend folder)
Step 4: Start your frontend server: npm run dev (in frontend folder)

5. DEPLOYMENT LINKING
=====================
When deploying to Vercel:
- Images in public folder are automatically served at: /images/products/...
- Database connects via MONGODB_URL environment variable
- Frontend connects to backend via NEXT_PUBLIC_API_URL environment variable

6. TIPS
=======
- All images should be in JPG format
- Recommended size: 800x1000 pixels for best quality
- Keep file sizes under 2MB for optimal loading
- Use descriptive, SEO-friendly filenames
